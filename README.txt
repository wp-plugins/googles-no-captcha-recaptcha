== Description ==

This plugin adds Google's No Captcha reCaptcha to WordPress comment, login, registration and lost password forms. A very short and sweet plugin which protects your website from bots trying to put spam comments, launch brute force attacks and creating fake accounts.

== Installation ==

1. Upload the google-no-captcha-recaptcha folder to the /wp-content/plugins/ directory.

2. Activate the plugin via the 'Plugins' menu in WordPress.

3. Plugin settings are located in "reCaptcha Options".
